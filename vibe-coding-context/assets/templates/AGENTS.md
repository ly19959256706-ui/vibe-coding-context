# Agent Instructions

Before non-trivial coding tasks, read `docs/context-index.md`.

Use `vibe-coding-context` for tasks that involve multiple files, UI flows, architecture, project documentation, or continuation across chats.

Do not read all project documents at once. Use `docs/context-index.md` to choose the relevant documents.

For ongoing work, maintain concise task memory in:

- `ai/task-brief.md`
- `ai/progress.md`
- `ai/findings.md`

Keep updates short. Record only goals, scope, decisions, blockers, useful findings, and verification results.
